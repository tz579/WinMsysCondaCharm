## get admin privilege
1. run "make me admin" and set the expiring time as long as possible if there is any

## update conda configurations
1. copy ".bashrc" into "C:\msys64\home\<your-username>" and replace the pre-existed one
2. right-click on the blank area of current folder that contains this instruction and choose "msys2 bash here"
3. run "mamba uninstall llvmlite numba librosa" and type "y" when any confirmation pops up
4. run "pip uninstall llvmlite numba librosa" and type "y" when any confirmation pops up
5. run "pip uninstall llvmlite numba librosa" again and type "y" when any confirmation pops up
6. run "pip cache purge" and then run "mamba clean --all -yf"
7. run "mamba update --all" and type "y" when any confirmation pops up
8. run "mamba env update --file env.yaml"
9. run "bash install.sh.conda.llvmlite"
10. run "bash install.sh.conda.numba"
11. run "bash install.sh.conda.librosa"
12. run "bash install.sh.conda.dllogger"
13. run "bash install.sh.conda.apex"
14. run "mamba clean --all -yf" and then run "pip cache purge"

## test current configurations
1. go to "C:\msys64\home\<your-username>" in windows explorer
2. right-click on the blank area and choose "msys2 bash here"
3. run "mkdir Work" and then run "cd Work"
4. run "git clone https://github.com/tz579/LJ_tacotron2.git"
5. run "cd LJ_tacotron2"
6. run "nohup bash script.1epoch.sh > script.1epoch.log &"
7. run "tail -f script.1epoch.log" to monitor the output
8. press "ctrl+x" whenever you want to stop monitoring the output
9. re-run step 7 whenever you want to monitor the output again
10. when everything is finished, go to output/ in windows explorer to check results
