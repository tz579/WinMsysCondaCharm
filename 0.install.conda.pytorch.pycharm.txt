## get admin privilege
1. run "make me admin" and set the expiring time as long as possible if there is any

## install & configure conda
1. download "Miniconda3 Windows 64-bit" from https://docs.conda.io/en/latest/miniconda.html
2. run "Miniconda3 Windows 64-bit" and choose "install for all users" when ask
3. search and right click "anaconda prompt" from start menu, press "run as Administrator"
4. run "conda config --set channel_priority strict"

## install & configure pytorch
1. cd to the dir where env.yaml exists
2. run "conda env update --file env.yaml"
3. run "pip3 install torch==1.10.2+cu113 torchaudio===0.10.2+cu113 -f https://download.pytorch.org/whl/cu113/torch_stable.html"
4. run "conda clean --all -yf" and then run "pip cache purge" to cleanup temp files

## install & configure pycharm
1. download "jetbrains-toolbox" from https://www.jetbrains.com/toolbox-app/
2. run "jetbrains-toolbox" and follow the instruction to install pycharm-community
3. search and open "pycharm-community" from start menu
4. press "create new project" and select "previously configured interpreter"
5. press "..." button on the right side of "previously configured interpreter"
6. select "conda environment" on the left side
7. press "..." button on the right side of "interpreter" and press "show hidden files and folders" on the opened window
8. find "C:\ProgramData\Miniconda3\python.exe" and press OK
9. press "..." button on the right side of "conda executable" and press "show hidden files and folders" on the opened window
10. find "C:\ProgramData\Miniconda3\scripts\conda.exe" and press OK
11. select "make available for all projects" and press OK
12. press OK again to finally open the new project
